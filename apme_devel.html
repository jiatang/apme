Build Instructions

APme was developed under Cygwin using GNU/C and the vim editor.

So, in order to successfully build APme you will need Cygwin with several development tools installed:

  • Go to the Cygwin Homepage and download and run the Cygwin installer (setup.exe)
  • During the installation phase, make sure to select the following packages:
    • mingw64-i686-gcc-core: the MinGW 32-bit compiler
    • make: the GNU Make tool
    • perl: the Perl interpreter, only if you wish to generate the release notes

After you're done, open up a shell window, change your current directory to the APme source code directory and simply type make. This should produce a win32 application in the src directory (src/APme.exe).

Dependencies

This tool require PCRE to compile properly. If not present, the build system will automatically download the PCRE sources from the internet and compile them.
