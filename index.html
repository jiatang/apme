APme: Abyss Point Tracker for Aion

Introduction

APme was the first automatic AP tracking tool for Aion: The Tower Of Eternity. It collects various statistics by reading the game client chat log, most importantly the abyss points gained for each group member. Based on this data, it can suggest who should loot the next AP relic so the AP is evenly spread between the group.

Features

  • Autoamtic AP sharing calculator
  • "Copy/Paste" interface; no need to leave the game to use it, the magic happens in your chat window!
  • Elyos <-> Asmodian translator w/o leaving the game
  • Built-in help system
  • Chat history
